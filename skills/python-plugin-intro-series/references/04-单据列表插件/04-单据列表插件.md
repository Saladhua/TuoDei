## 一、列表插件简介

- 表单插件只在单据编辑界面触发，**单据列表不会触发表单插件**，因此需要列表插件处理列表相关功能
- **Web服务层插件**，依赖单据列表界面(ListView)触发
- 应用于单据列表、基础资料列表
- C#基类：`AbstractListPlugIn`
- 列表插件与表单插件都是界面类插件，有共同父类 `AbstractDynamicFormPlugIn`，因此表单插件的部分用法在列表插件中同样适用

## 二、列表插件重要成员介绍

`this.Context` 用法与表单插件一致，不再重复；以下为列表插件特有的成员属性。

### ListView — 列表视图层对象

```python
# 当前列表对应的单据标识
```

## 三、列表插件常用事件介绍

### 事件一览表

| 事件 | 用途 |
|------|------|
| `PrepareFilterParameter` | 干预列表过滤条件(最常用) |
| `BarItemClick` | 菜单点击时触发(可取消) |
| `AfterBarItemClick` | 菜单点击后触发(实现功能) |
| `ListRowDoubleClick` | 列表行双击 |
| `EntryButtonCellClick` | 单元格超链接点击 |
| `OnFormatRowConditions` | 条件格式化(设置行颜色) |
| `BeforeDoOperation` | 批量操作前 |
| `AfterDoOperation` | 批量操作后 |

### PrepareFilterParameter — 列表过滤事件

加载列表时触发，可干预过滤条件。

```python
def PrepareFilterParameter(e):
    custfilterObj = e.CustomFilter
    filterStr = e.FilterString
    sortStr = e.SortString
    statusFilterStr = e.StatusFilterString
    myFilterStr = ("FBillNo like '%{0}%' ").format("1")
    e.AppendQueryFilter(myFilterStr)
```

### AfterBarItemClick — 列表菜单点击后事件

菜单点击完成后触发，自定义菜单功能建议在此实现。

```python
def AfterBarItemClick(e):
    key = e.BarItemKey.ToUpperInvariant()
    if (key <> "TestBtn".ToUpperInvariant()):
        return
    selectedRowsInfo = this.ListView.SelectedRowsInfo
    billIDs = selectedRowsInfo.GetPrimaryKeyValues()
    entryIDs = selectedRowsInfo.GetEntryPrimaryKeyValues()
    datas = this.ListModel.GetData(selectedRowsInfo)
    if (datas.Count <= 0):
        this.View.ShowWarnningMessage("未选择任何行！")
        return
    msg = ("选中了[{0}]个单据,[{1}]条明细！").format(billIDs.Length, entryIDs.Length)
    s = ""
    for row in datas:
        billNo = row["FBILLNO"]
        billId = row["FID"]
        s = ("{0}\r\n{1}--{2}").format(s, billNo, billId)
        if (row.DynamicObjectType.Properties.ContainsKey("FMaterialId_Id")):
            matId = row["FMaterialId_Id"]
            s = ("{0}--{1}").format(s, matId)
        if (row.DynamicObjectType.Properties.ContainsKey("FMaterialId_Ref")):
            matObj = row["FMaterialId_Ref"]
            matNum = matObj["Number"]
            matName = matObj["Name"]
            s = ("{0}--{1}--{2}").format(s, matNum, matName)
        if (row.DynamicObjectType.Properties.ContainsKey("FDATE")):
            date = row["FDate"]
            s = ("{0}--{1}").format(s, date)
    msg = ("提示：{0}").format(s)
    this.View.ShowMessage(msg)
```

### ListRowDoubleClick — 列表双击事件

```python
def ListRowDoubleClick(e):
    row = e.Row
    colKey = e.ColKey
    currentPageRowsInfo = this.ListView.CurrentPageRowsInfo
    startRow = this.ListModel.StartRow
    num = row - startRow
    listSelectedRow = currentPageRowsInfo[num - 1]
    FID = listSelectedRow.PrimaryKeyValue
    entryId = listSelectedRow.EntryPrimaryKeyValue

    queryParam = QueryBuilderParemeter()
    queryParam.FormId = this.View.BillBusinessInfo.GetForm().Id
    queryParam.BusinessInfo = this.View.BillBusinessInfo
    fldKey = colKey
    if ("." in colKey):
        colItem = SelectorRefItemInfo(colKey)
        fldKey = colKey.Replace(".", "_")
        colItem.PropertyName = fldKey
        queryParam.SelectItems.Add(colItem)
    else:
        queryParam.SelectItems.Add(SelectorItemInfo(colKey))
    entity = this.View.BillBusinessInfo.GetEntity(listSelectedRow.EntryEntityKey)
    if (entity <> None):
        queryParam.FilterClauseWihtKey = ("{0}_{1}={2}").format(entity.Key, entity.EntryPkFieldName, entryId)
    else:
        queryParam.FilterClauseWihtKey = ("{0}={1}").format(this.View.BillBusinessInfo.GetForm().PkFieldName, FID)
    dataRows = QueryServiceHelper.GetDynamicObjectCollection(this.Context, queryParam)
    colValue = ""
    if (dataRows.Count > 0):
        colValue = ("{0}").format(dataRows[0][fldKey])
    msg = ("双击了第{0}行的[{1}]").format(row, colValue)
    this.View.ShowMessage(msg)
    e.Cancel = True
```

### OnFormatRowConditions — 列表条件格式化事件

可根据条件动态设置行颜色。

```python
def OnFormatRowConditions(args):
    if (args.DataRow.ColumnContains("FMaterialId_Ref")):
        matObj = args.DataRow.DynamicObject["FMaterialId_Ref"]
        matNum = ("{0}").format(matObj["Number"])
        if (matNum[0:1] == "1"):
            fc = FormatCondition()
            fc.ForeColor = "#FFFF9B98"
            args.FormatConditions.Add(fc)
    if (args.DataRow.ColumnContains("FDate")):
        dateStr = str(args.DataRow["FDate"])
        date = DateTime.Parse(dateStr).ToString("yyyy-MM-dd")
        if (date == DateTime.Now.ToString("yyyy-MM-dd")):
            fc = FormatCondition()
            fc.BackColor = "#0000FF"
            args.FormatConditions.Add(fc)
```

### BeforeDoOperation — 批量操作前

```python
def BeforeDoOperation(e):
    opCode = e.Operation.FormOperation.Operation.ToUpperInvariant()
    if (opCode == "DELETE"):
        e.Cancel = True
        this.View.ShowWarnningMessage("取消删除！")
```

```python
def AfterDoOperation(e):
    opCode = e.Operation.Operation.ToUpperInvariant()
    this.View.ShowMessage(str(opCode))
```

## 五、列表插件读取单据数据

列表插件内不能直接读取完整单据数据包，需要变通获取。一般思路：读取勾选数据行 → 构建过滤条件 → 查询单据。

### 常用成员回顾

```python
currentSelectedRow = this.ListView.CurrentSelectedRowInfo   # 焦点行(单行)
selectedRowsInfo = this.ListView.SelectedRowsInfo            # 勾选数据集合
billIDs = selectedRowsInfo.GetPrimaryKeyValues()             # 单据ID数组
entryIDs = selectedRowsInfo.GetEntryPrimaryKeyValues()       # 明细ID数组
datas = this.ListModel.GetData(selectedRowsInfo)             # 数据包(DynamicObjectCollection)
if (datas.Count <= 0):
    this.View.ShowWarnningMessage("未选择任何行！")
    return
currentPageRowsInfo = this.ListView.CurrentPageRowsInfo     # 当前页数据
startRow = this.ListModel.StartRow                           # 当前页开始序号
```
